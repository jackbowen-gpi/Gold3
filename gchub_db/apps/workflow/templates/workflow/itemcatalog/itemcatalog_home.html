Item Catalog



The Item Catalog tracks specific sizes/products. This will be general information about that size (ie. what type of item it is). Item Catalog entries marked as active are what populates the pulldown menu of available items in the job.

Item Specifications



Item Specs track the varied manufacturing specifications for each item in the Item Catalog (ie. which plants print this size and on which presses).

Example: Item Catalog stores that an SMR-16 is a Foodservice item that is a Hot Cup. Item Specs stores that we print that cup on the Kenton Webtron and it can print with 6 colors.

Step & Repeat Specifications



Step & Repeat Specs track the special manufacturing processes used to convert the cups (ie. roll fed, blank fed, big cylinder, small cylinder, etc).

Example: Item Catalog stores that an SMR-16 is a Foodservice item that is a Hot Cup. Item Specs stores that we print that cup on the Kenton Webtron. Step and Repeat Spec stores that it is a roll fed configuration and it can print with 6 colors. Step and Repeat spec also tells you the rectangle dimension for the straight art and the number around, number across and total number of blanks.
